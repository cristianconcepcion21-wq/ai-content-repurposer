# 🎥 AI Content Repurposer

A Streamlit app that turns YouTube videos into:
- Blog posts ✍️
- LinkedIn posts 💼
- Tweet threads 🐦

## 🚀 How to Run
1. Clone repo
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run:
   ```bash
   streamlit run app.py
   ```

## 🌐 Deployment
Deploy easily on [Streamlit Cloud](https://streamlit.io/cloud).
